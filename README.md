# Manuscript_2026
scRNA-seq analysis code
This repository contains the code used for the single-cell analysis of Cd45 positive cells from MMTV-pymt mice.

Data and Folder Structure:
    Code Files: All code files, annotated for each figure, can be found in the "/Source" folder.

Requirements:

Software Environment
    Operating System: Windows 11 with WSL2 (Ubuntu 22.04.5 LTS) on x86_64
    Python Version: 3.11.11 (conda-forge)

Python Packages:
    scanpy 1.10.4
    pandas 2.2.2
    scipy 1.15.1
    mygene 3.2.2
    numpy 1.26.4
    matplotlib 3.10.0
